Значения (Литералы) | Values (Literals)

Определение

  • Значения (Литералы) — сырые данные (raw data) присваиваемые переменной или константе в языках программирования.

Пример литерала

  • int variable = 123;, где int — тип данных, variable — переменная, 123 — значение (литерал).

Виды литералов в Python

  1. Строковые литералы (String Literals) — последовательность символов окруженная одинарными, двойными или тройными кавычками.
    1. Однострочный строковый литерал (single line strings) — заканчивается при обнаружении символа новой строки и может быть определен путем заключения одного или нескольких символов внутри одинарных или двойных кавычек.
    1. Многострочный строковый литерал (multiline strings) — представляет собой несколько строк заключенных в тройные кавычки.
  1. Числовые литералы (Numeric Literals)
    1. Целые числа (Integers) — числа без дробного компонента.
      1. Целое число в десятичной системе счисления. myNum = 123
      1. Целое число в двоичной системе счисления (начинается с “0b” и состоит из цифр 0 и 1). myNum = 0b1111011
      1. Целое число в шестнадцатеричной системе счисления (начинается с “0х”). myNum = 0x1E0F3
      1. Целое число в восьмеричной системе счисления (начинается с “0o”). myNum = 0o173
    1. Вещественные числа (Floating Point Numbers. числа с плавающей точкой) — число состоящее из целой и дробной частью, разделенные точкой. myNum = 123.345
    1. Комплексные числа (Complex Numbers) — числа в виде a+bj, где a представляет собой реальную часть, b представляет собой мнимую часть. myNum =3+4j
  1. Логические литералы (Boolean Literals)
    1. Истина (True) myVar1 = True
    1. Ложь (False) myVar2 = False
  1. Специальный литерал None (Special Literal None) — указывает, что переменная на которую назначен это литерал не ссылается (refer to) на какой-либо объект. myObj = None
  1. Коллекции литералов (Literal Collections) — структуры для хранения множества литералов.
    1. Список (List) — позволяет хранить множество элементов с различными типами данных. myList=[1,2,3,"abc",3.14]
    1. Кортеж (Tuple) — позволяет хранить множество элементов с различными типами данных, являются неизменяемым и не позволяет добавить новые значения. myTuple=(1,2,3,"abc",3.14)
    1. Словарь (Dictionary) — позволяет хранить множество элементов в формате ключ: значение.
    1. Набор (Set) — позволяет хранить множество неупорядоченных и уникальных элементов (некий массив данных) с различными типами данных, может иметь внутри себя вложенный список. С каждым элементом списка можно взаимодействовать по отдельности (например, выводить на экран). mySet={1,2,3,"pfb",3.14}

Что не относится к значениям (литералам)

  1. Идентификатор (identifier) — ссылка на значение в памяти, может быть имя переменной. std::cout
  1. Утверждение (объявление переменной) x = 0
  1. Выражение 1+2
