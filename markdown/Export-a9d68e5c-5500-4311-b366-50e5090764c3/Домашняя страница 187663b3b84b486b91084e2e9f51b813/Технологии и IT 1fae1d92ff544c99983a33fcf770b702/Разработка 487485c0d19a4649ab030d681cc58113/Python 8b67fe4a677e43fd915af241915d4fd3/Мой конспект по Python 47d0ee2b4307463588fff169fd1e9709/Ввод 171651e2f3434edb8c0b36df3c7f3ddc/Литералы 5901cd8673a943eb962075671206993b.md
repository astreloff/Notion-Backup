# Литералы

**Определение**

Литералы — сырые данные (raw data) присваиваемые переменной или константе в языках программирования.

**Виды литералов в Python**

1. Строковые литералы (String Literals) — последовательность символов окруженная одинарными, двойными или тройными кавычками.
2. Цифровые литералы (Numeric Literals)
3. Логические литералы (Boolean Literals)
4. Специальный литерал None (Special Literal None)
