# Рекомендации

1. **Руки не должны забиваться.**
Если это происходит, значит неправильно распределена нагрузка.
2. Напрячь плечи, стараясь опустить их вниз, а не висеть "как сосиска".
3. Тянуться грудью к турнику, представлять, будто хочу сомкнуть локти за спиной, подтянуть локоть к пояснице.
