# Выражение else

Вывод: Перебрали всю строку и не нашли букву 'ч’
Категория: Циклы
Текст программы: for i in 'Если в строке найдется заданный символ, цикл остановится':
    if i == 'ч':
        break
    else:
        print('Перебрали всю строку и не нашли букву \'ч\'')

Выражение `else` в цикле (`for` или `while`) проверяет, был ли произведен выход из цикла инструкцией `break`, или цикл завершился при выполнении.
