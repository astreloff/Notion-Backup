<aside>
ℹ️ В отличие от других языков, в Python нет объявления переменных (Variable Declaration), поэтому для создания переменной мы присваиваем ей значение (assign value).
Переменная создается в момент присвоения ей значения.

</aside>

- Python не относится языкам со строгой типизацией, т.е. при создании/изменении переменной не обязательно указывать тип данных.
- Каждая переменная использует только один тип данных.
- При изменении значения переменной, тип данных меняется автоматически в зависимости от указанного значения.
- Переменные принимают на вход и позволяю хранить значения (values).
- В переменную можно установить пустое значение. Например: `x = None`
- Название переменной может быть как коротким (`x`, `y`), так и длинным (`first_name`, `last_name`).
- Название переменной должно начинаться с буквы или нижнего подчеркивания, **но** не с цифры.
- Название переменной может содержать только буквенно-цифровые символы и символы подчеркивания (а-z, 0-9 и _ ).
- Название переменных чувствительны к регистру (age, Age, AGE — три разных переменных).
