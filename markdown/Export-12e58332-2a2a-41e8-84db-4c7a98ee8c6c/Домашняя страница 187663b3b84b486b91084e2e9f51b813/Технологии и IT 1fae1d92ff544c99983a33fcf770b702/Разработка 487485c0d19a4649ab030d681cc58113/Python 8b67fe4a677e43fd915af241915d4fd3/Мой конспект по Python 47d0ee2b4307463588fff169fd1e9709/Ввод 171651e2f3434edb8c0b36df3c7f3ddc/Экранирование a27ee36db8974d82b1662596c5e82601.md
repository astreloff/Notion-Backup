# Экранирование

**Символ экранирования (Escape character)**
Символ косой черты `\` используется для экранирования символов в коде и ставится перед символом.
Если поставить `\` перед кавычкой (одинарной или двойной), это скажет интерпретатору о том, что кавычку нужно рассматривать не как начало или конец строки, а как часть строки.

**Экранирование последовательности (Escape sequence)**

1. Конструкция `\n` используется для переноса на новую строку. Не является видимой частью строки. Рекомендуется к использованию вместо `\r` из-за проблем трактовки в разных кодировках.
2. Табуляция `\t` (разрыв, получаемый при нажатии на кнопку Tab)
3. Возврат каретки `\r` для перемещения курсора в начало строки, не перемещая его в новую строку (только в Windows).
