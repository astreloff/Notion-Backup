# Сильная/строгая типизация (Strong typing)

Определение: Свойство некоторых языков программирования (например, Python) не позволяющее совершать операции с разными типами данных. Например, нельзя выполнить print(’7’+3), т.к. 7 является строкой, а 3 является числом.

В таких языках как JavaScript, PHP и т.п. осуществляется автоматическое неявное преобразование (например Число 1 + Строка 7 = Строка 17).
