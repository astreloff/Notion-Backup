# Индекс (Index)

**Индекс** — позиция элемента внутри массива. Например, символа внутри строки. 

- Индексы начинаются с 0, поэтому для получения первого символа, нужно указать индекс `0`.
- Индекс последнего элемента равен длине строки минус единица, поэтому для получения последнего символа нужно сказать индекс `-1`.

```python
first_name = 'Alexander'
print(first_name[1])
# => l

first_name = 'Alexander'
print(first_name[-1])
# => r

first_name = 'Alexander'
index = 0
print(first_name[index])
# => A
```
