# Sampler convergence

Определение: Сходимость методов выборки подразумевает то, что после определенного количества шагов выборки (sampling steps) генерируемое изображение перестает меняться и увеличение шагов не приводит к изменениям изображения. Меньше изменений видно по мере увеличения количества шагов.
Примечание: Чем меньше количество шагов для генерации изображения, которое не будет изменяться, тем лучше. Это позволяет при генерации изображений пачками сократить время, не используя большое количество шагов.
