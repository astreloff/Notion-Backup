# hypernetwork

Определение: Гиперсеть
1. HyperNetwork is a network that generates weights for a main network
2. Is a way to train an addon, which you can load or unload to an model
3. Позволяет корректировать стиль генерируемого изображения, например anime hypernetwork делает больший уклон на изображения в стиле аниме.
