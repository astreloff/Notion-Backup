# Инженер (Engineer)

Описание: Урон: Гаечные ключи (5 уровня) + Взрывная турель + Диплом + Книга + Шестеренка + Металлодетектор
Защита: HP + Реген + Броня + Хилящие турели
Дополнительно: Удача + Сбор
Примечание: Мощный билд, позволяющий практически стоять на месте, стоит взять рубашку лесоруба, т.к. сложно разбивать ящики гаечными ключами и удаляться от туррелей.
Оценка: ⭐⭐⭐
Прохождений: 2
