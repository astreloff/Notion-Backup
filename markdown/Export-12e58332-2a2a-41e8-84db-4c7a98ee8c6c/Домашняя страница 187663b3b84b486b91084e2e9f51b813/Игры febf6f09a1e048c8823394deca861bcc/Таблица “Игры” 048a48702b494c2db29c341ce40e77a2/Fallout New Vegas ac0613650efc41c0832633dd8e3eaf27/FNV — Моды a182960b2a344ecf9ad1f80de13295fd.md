1. Запустить игру для генерации `.ini` файлов.
2. Установить русификатор текстур (FNV-Localization). ([Ссылка](https://steamcommunity.com/sharedfiles/filedetails/?id=565838905))
    1. Вшить локализацию из `.esp` файлов игры и DLC в `.esm` файлы ([Ссылка](https://modgames.net/publ/obuchajushhie_stati/obshhaja/1/12-1-0-2582)) **ИЛИ** скачать готовые `.esm` файлы ([Ссылка](https://bethplanet.ru/fallout-new-vegas/mods/patches/rusifikaciya-fallout-new-vegas-i-osnovnyh-dlc)).
3. Установить русскую озвучку, скопировав распакованные файлы в корень игры.
4. Выполнить инструкцию по установке модов.
