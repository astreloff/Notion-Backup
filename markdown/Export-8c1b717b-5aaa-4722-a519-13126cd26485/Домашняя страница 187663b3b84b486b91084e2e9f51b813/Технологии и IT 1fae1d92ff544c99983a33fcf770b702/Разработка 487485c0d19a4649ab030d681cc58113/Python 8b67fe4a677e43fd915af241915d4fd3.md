3. Книга “Грокаем алгоритмы (Адитья Бхаргава)” ([Ссылка](https://drive.google.com/file/d/1hA12e975vCQLimCkdtXK94BGD439hDym/view?usp=sharing))
    1. Алгоритмы и структуры данных ([Ссылка](https://github.com/trekhleb/javascript-algorithms/blob/master/README.ru-RU.md))
4. Математика с нуля ([Ссылка](http://spacemath.xyz/))
5. Решение реальных задач с использованием Python (решить 100+)
[Список математических задач для решения на ЯП ([Ссылка](http://euler.jakumo.org/))](Python%208b67fe4a677e43fd915af241915d4fd3.md)
6. Книга “Изучаем Python (Лутц Марк)” (см. последний том)
7. Книга “Программирование на Python (Лутц Марк)” (см. последний том)
8. Пишем Telegram-ботов на Python (v2) ([Ссылка](https://mastergroosha.github.io/telegram-tutorial-2/))
9. Изучение библиотек и фреймворков Python (Курируемый лист фрейморков и т.п.) ([Ссылка](https://github.com/vinta/awesome-python))
    1. Библиотека Requests
    2. Scrapy — фреймворк для веб-скрапинга.
    3. Django — свободный фреймворк для веб-приложений на языке Python, использующий шаблон проектирования MVC.
    4. WxPython — обёртка библиотеки кроссплатформенного графического интерфейса пользователя для Python.
    5. QLAlchemy — это программная библиотека на языке Python для работы с реляционными СУБД с применением технологии ORM. Служит для синхронизации объектов Python и записей реляционной базы данных.
    6. Beautiful Soup - это пакет Python для анализа документов HTML и XML. Он создает дерево синтаксического анализа для проанализированных страниц, которое можно использовать для извлечения данных из HTML, что полезно для парсинга веб-страниц.
    7. Pygame — набор модулей языка программирования Python, предназначенный для написания компьютерных игр и мультимедиа-приложений.Sphinx — это генератор документации, который преобразует файлы в формате reStructuredText в HTML website и другие форматы.
