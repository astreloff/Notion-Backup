# Подготовка к работе

## Установка интерпретатора Python

1. Скачать последнюю версию интерпретатора Python с [официального сайта](https://www.python.org/downloads/).
2. Поставить галочку “Add Python to PATH” для добавления каталога установки в список системных переменных PATH. Это позволит использовать терминал Windows для запуска интерпретатора (команды **py**, **python** ) и ****для запуска установщика пакетов (**pip**).
    1. Путь установки по умолчанию: C:\Users\User\AppData\Local\Programs\Python\Python310
