# Условные выражения

### Операции сравнения

<aside>
ℹ️ Операции сравнения могут сравнивать различные объекты - строки, числа, логические значения, однако оба операнда операции должны представлять **один и тот же [тип данных](../%D0%94%D0%B0%D0%BD%D0%BD%D1%8B%D0%B5%204b19bbf843c748bca35b57c499f83ae1/%D0%A2%D0%B8%D0%BF%D1%8B%20%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D1%85%20Data%20Types%20bf4d21c3b60a46d0abcc8b3c22f8907f.md)**.

</aside>

- **`==` —** Возвращает True, если оба операнда равны. Иначе возвращает False.
- **`!=` —** Возвращает True, если оба операнда НЕ равны. Иначе возвращает False.
- **`>`** (больше чем) — Возвращает True, если первый операнд больше второго.
- **`<`** (меньше чем) — Возвращает True, если первый операнд меньше второго.
- **`>=`** (больше или равно) — Возвращает True, если первый операнд больше или равен второму.
- **`<=`** (меньше или равно) — Возвращает True, если первый операнд меньше или равен второму.

### Логические операции

<aside>
ℹ️ Если один из операндов оператора `and` возвращает `False`, то другой операнд уже не оценивается, так как оператор в любом случае возвратит `False`.
Если один из операндов оператора `or` возвращает `True`, то второй операнд не оценивается, так как оператор в любом случае возвратит `True`.
Подобное поведение позволяет немного увеличить производительность, т.к. не приходится тратить ресурсы на оценку второго операнда.

</aside>

- `and` (логическое умножение) — Возвращает True, если оба выражения равны True.
- `or` (логическое сложение) — Возвращает True, если хотя бы одно из выражений равно True.
- `not` (логическое отрицание) — Возвращает True, если выражение равно False.

### Условные операторы

- Оператор `if` — срабатывает при выполнении заданных условий.
- Оператор `elif` — дополнительное условие проверки, при выполнении которого оператор `else` не срабатывает.
- Оператор `else` — используется в связке с `if` и срабатывает если один или несколько `if` до него не сработали.

**Пример кода:**

```python
user_data = int(input('Введите число от 1 до 3: '))
param1 = False
if param1 and user_data == 1:
    print('Число 1, param = True')
elif param1 and user_data == 2:
    print('Число 2, param = True')
elif param1 and user_data == 3:
    print('Число 3, param = True')
elif not param1 and user_data == 1:
    print('Число 1, param = False')
elif not param1 and user_data == 2:
    print('Число 2, param = False')
elif not param1 and user_data == 3:
    print('Число 3, param = False')
else:
    print('Число вне диапазона от 1 до 3')
```
