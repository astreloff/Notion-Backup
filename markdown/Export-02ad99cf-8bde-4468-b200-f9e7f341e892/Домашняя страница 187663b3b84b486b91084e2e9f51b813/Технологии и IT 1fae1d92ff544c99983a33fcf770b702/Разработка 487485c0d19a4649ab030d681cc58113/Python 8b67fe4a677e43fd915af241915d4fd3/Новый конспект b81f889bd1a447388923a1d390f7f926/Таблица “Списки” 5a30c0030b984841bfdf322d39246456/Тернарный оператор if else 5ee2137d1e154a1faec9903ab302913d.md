# Тернарный оператор if/else

Вывод: Введите число от 1 до 10. Число: 5
Верно!
Категория: Ветвление
Текст программы: number = int(input("Введите число от 1 до 10. Число: "))
answer = 'Верно!' if 1 <= number <= 10 else 'Число вне заданного диапазона!'
print(answer)

> Синонимы: трехместное выражение if/else.
