# Пацифист (Pacifist)

Описание: Защита: Ладони + Шокер + HP + Реген + Уклонение + Броня + Отталкивание (боксерская перчатка) + замедление (робот)
Дополнительно: Скорость + Дальность + Удача + Сбор
Примечание: Сложный билд, имеет потенциал на высоких уровнях при большом количестве хп, регенерации и брони.
Прохождений: 1
