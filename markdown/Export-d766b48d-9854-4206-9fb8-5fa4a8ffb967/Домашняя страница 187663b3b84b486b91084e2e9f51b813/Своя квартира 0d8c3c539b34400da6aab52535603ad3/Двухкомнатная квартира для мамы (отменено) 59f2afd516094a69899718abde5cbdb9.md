# Двухкомнатная квартира для мамы (отменено)

---

# Ипотека (двушка для нас)

1. **Срок ипотеки**: 10-12 лет.
2. **Стоимость квартиры**: 12-13 м. руб.
3. **Ежемесячный платеж**: 100-110 т. руб.
4. **Подушка безопасности**: 150-200 т. руб.

# Пожелания

1. **Двухкомнатная** квартира.
2. Рассматриваем только **новостройки**, а не вторичку.
3. Большая **кухня**.
4. Раздельный **санузел**.
5. **Этаж** не ниже 5 `изучить тему`.
6. Наличие **грузового лифта**.

# TO-DO

1. Накопить на первичный взнос.
2. Собрать информацию.
    1. Узнать у работодателя про ипотеку и индексацию ЗП.
    2. Пообщаться с родителями (мама, дядя Насти, папа, бабушка).
3. Выбрать застройщика и жилье.
4. Выбрать банк и процент вклада.

# Изучить

[sberbank.ru](http://www.sberbank.ru/ru/person/credits/homenew)

Варианты ипотеки

- Рефинансирование, реструктуризация, перекредитование
- Варианты застройщиков и их условия (ПИК и т.п.).
