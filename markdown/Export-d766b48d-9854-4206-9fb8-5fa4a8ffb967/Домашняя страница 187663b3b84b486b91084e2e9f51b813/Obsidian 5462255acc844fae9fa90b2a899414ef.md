| Plugin Name | Description | Plugin Link |
| --- | --- | --- |
| Dataview | Allows you to create custom queries to filter and sort your notes based on metadata and data attributes. | https://github.com/blacksmithgu/obsidian-dataview |
| QuickAdd | Allows you to create new notes and append/prepend content to existing notes from a single command palette. | https://github.com/chhoumann/obsidian-quickadd |
| Templater | Allows you to create templates for your notes, which can include placeholders that will be automatically filled in when creating a new note. | https://github.com/SilentVoid13/Templater |
| Sortable | Allows you to manually sort your notes within a folder or tag view. | https://github.com/alexandru-dinu/obsidian-sortable |
| Supercharged Links | Enhances links in your notes by adding features like bidirectional linking, automatic backlinks, and link previews. | https://github.com/deathau/supercharged-links |
| Paste URL into selection | Allows you to paste a URL into a selected piece of text, automatically creating a hyperlink. | https://github.com/alienmantech/obsidian-paste-url-into-selection |
| Typing Transformer | Allows you to create text transformation rules, which will automatically replace specific strings with other strings as you type. | https://github.com/kepano/obsidian-typing-transformer |
| Various Complements | A collection of small but useful plugins, including Block Reference Plus, Collapse All Folders, and Drag and Drop. | https://github.com/shichongrui/obsidian-various-complements |
| LanguageTool Integration | Provides grammar and style checking for your notes using the LanguageTool service. | https://github.com/argenos/obsidian-languagetool |
| Tag Wrangler | Provides various tools for managing tags, including renaming, merging, and auto-completion. | https://github.com/pjeby/tag-wrangler |
| Table of Contents | Generates a table of contents for your note based on headers and sub-headers. | https://github.com/ekomizer/obsidian-toc |
| Text Format | Provides various text formatting tools, including title case, sentence case, and remove formatting. | https://github.com/akosbalasko/obsidian-text-format |
| Clear Unused Images | Removes images that are not used in any of your notes, freeing up space on your hard drive. | https://github.com/mrjackphil/obsidian-clear-unused-images |
| Ozan's Image in Editor Plugin / Obsidian Image Toolkit | Provides various image editing tools, including resizing, rotating, and cropping. | https://github.com/ozntel/obsidian-image-in-editor |
| Text Snippets | Provides a library of commonly used text snippets that can be inserted into your notes with a few keystrokes. | https://github.com/bjsi/obsidian-snippets |
| Copy Image and URL context menu | Adds a context menu option to copy the image and its URL at once. | https://github.com/TfTHacker/obsidian-copy-image-and-url-context-menu |
| MySnippets | Provides a library of personal text snippets that can be inserted into your notes with a few keystrokes. | https://github.com/rl-king/obsidian-mysnippets |
| nomnoml Diagram / D2 | Allows you to create and render diagrams in your notes using the nomnoml language. | https://github.com/argenos/nomnoml-diagram |
| Mousewheel Image Zoom | Allows you to zoom in and out of images in the editor using the mouse wheel. | https://github.com/chhoumann/ |
