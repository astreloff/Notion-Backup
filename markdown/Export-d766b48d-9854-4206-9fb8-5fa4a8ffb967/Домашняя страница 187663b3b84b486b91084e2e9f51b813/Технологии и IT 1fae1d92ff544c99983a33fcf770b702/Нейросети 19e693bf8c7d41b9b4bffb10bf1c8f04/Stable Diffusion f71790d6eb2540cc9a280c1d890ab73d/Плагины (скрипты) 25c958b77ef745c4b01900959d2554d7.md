# Плагины (скрипты)

### [Booru tag autocompletion](https://github.com/DominikDoom/a1111-sd-webui-tagcomplete)

Отображает подсказки автозаполнения для тегов с имиджбордов, вроде Danbooru. Использует файлы .CSV с тегами и включает файл конфигурации для настройки.

Поскольку некоторые модели Stable Diffusion были обучены с использованием этой информации, например [Waifu Diffusion](%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D0%B8%20%D0%B8%20%D0%BF%D1%80%D0%B5%D0%B4%D1%81%D1%82%D0%B0%D0%B2%D0%BB%D0%B5%D0%BD%D0%B8%D1%8F%203dd4646932db4909a9851d1fc07505f0/%D0%A2%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D0%B0%20%E2%80%9C%D0%9C%D0%BE%D0%B4%D0%B5%D0%BB%D0%B8%E2%80%9D%20dd42e5bd856840679bcd80c21e470ff9/Waifu%20Diffusion%20d8fa438a8df445369f10d428fdd31d6b.md) , использование точных тегов в запросе часто может улучшить композицию и помочь добиться желаемого вида.
