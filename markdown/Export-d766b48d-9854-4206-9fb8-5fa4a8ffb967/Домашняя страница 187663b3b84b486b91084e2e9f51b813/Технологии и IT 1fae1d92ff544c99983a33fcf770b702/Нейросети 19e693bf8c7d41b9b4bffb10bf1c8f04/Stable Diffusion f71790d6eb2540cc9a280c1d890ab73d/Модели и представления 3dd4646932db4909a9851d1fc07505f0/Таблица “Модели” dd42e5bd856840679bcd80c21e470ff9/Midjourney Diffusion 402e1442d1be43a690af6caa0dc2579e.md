Ссылка: https://huggingface.co/prompthero/midjourney-v4-diffusion
Стили: All

# Описание

Модель Dreambooth для переноса стиля с одного изображения на другое (аналогично функции Remix в Midjourney v4).

Использовать тег `mdjrny-v4 style` в запросе. **Пример:** `mdjrny-v4 style <другие_теги>`.
