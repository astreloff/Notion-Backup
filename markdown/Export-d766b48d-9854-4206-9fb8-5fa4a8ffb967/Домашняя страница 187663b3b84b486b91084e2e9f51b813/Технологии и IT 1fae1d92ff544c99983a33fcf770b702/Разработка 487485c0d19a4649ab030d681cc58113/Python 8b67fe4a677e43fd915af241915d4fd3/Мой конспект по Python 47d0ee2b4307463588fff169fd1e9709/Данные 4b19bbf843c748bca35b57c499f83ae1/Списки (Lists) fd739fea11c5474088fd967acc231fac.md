# Списки (Lists)

- Список может хранить множество различных значений (некий массив данных), в остальном похож на переменённую.
- Значения в списке могут иметь разные типы данных.
- Список может иметь внутри себя вложенный список.
- С каждым элементом списка можно взаимодействовать по отдельности (например, выводить на экран).
