# Циклы и операторы в них

<aside>
ℹ️ Хорошей практикой для именования идентификаторов счетчика является использование имен перемененных **`i`** (от итерации), **`j`**, **`k`** и так далее, при необходимости. Где `i` означает предельное значение счетчика циклов, при котором происходит выход из цикла, `j` — внутренний счетчик следующего цикла, и т. д.

</aside>

### Цикл for

- Цикл выполняется, пока не дойдет до конца в условии.
- Удобно использовать для перебора или создания диапазона [range()](../%D0%A4%D1%83%D0%BD%D0%BA%D1%86%D0%B8%D0%B8%209e3a832939aa4b4fb0f9861c2d66b776/range()%200f3109e5e6a1483db6e7c035f40b3187.md) .

**Пример:**

```python
# Подсчет количества встреченных символов в строке.
word = 'Hello world!'
count = 0
for i in word: #вместо переменной можно указывать строку для перебора
    if i == 'l':
        count += 1
print('Count: ', count)
```

### Цикл while

- Цикл выполняется, пока условие истинно.

**Пример:**

```python
i = 5
while i <= 15:
    print(i)
    i+=2
```

### Операторы циклов

### Brake

- Оператор `Brake` — Позволяет выйти из цикла при соблюдении условия.

**Пример:**

```python
for i in range (1,10):
    if i == 5:
        break
    print(i)
```

### Continue

- Оператор `Сontinue` — Позволяет пропустить итерацию выполнения цикла.

**Пример:**

```python
for i in range (1,10):
    if i == 5:
        continue
    print(i)
```
