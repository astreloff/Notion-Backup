# Выбор элементов по тексу (#text)

<aside>
<img src="https://www.notion.so/icons/info-alternate_gray.svg" alt="https://www.notion.so/icons/info-alternate_gray.svg" width="40px" /> `[1]` указывает на то, что необходимо вернуть первый найденный элемент.

</aside>

Для получения элемента по тексту содержащемуся внутри можно использовать следующий селектор: `div[@class="coordsAgence"]/text()[1]`
