# range()

Вид: Операции с числами
Входные параметры: range([start],stop,[step])
start — начальное значение,
stop — конечное значение,
step — шаг.

Можно не указывать первый параметр, тогда цикл начнется с 0.
Выходящее значение: int, последовательность целых чисел, с заданным шагом
Описание: Генерирует серию целых чисел, от значения start до stop.
Можно использовать в цикле for для обхода диапазона как списка.
Пример: for i in range(1,4):
print(i) = 1 2 3
